# Design Context — 풀다 AI

## Target Audience
- 한국 중·고등학생 (13-18세): 수업 후 자기주도 학습
- 한국 수학 교사: 오개념 진단 + 보충수업 설계

## Use Cases
- 학생: 수학 문제에서 막혔을 때 AI에게 질문 → 소크라틱 질문으로 스스로 해결
- 교사: 학생들이 어디서 막히는지 히트맵으로 파악 → 다음 수업에 반영

## Brand Personality / Tone
- **신뢰감 있는 교육 도구** — 장난감이 아닌 전문 도구
- **차분하고 명확한** — 학생이 집중할 수 있는 환경
- **따뜻하지만 절제된** — 친절하되 산만하지 않음

## Aesthetic Direction
- **Tone**: 리파인드 미니멀리즘 (refined minimalism) — 여백과 타이포그래피로 위계 형성
- **NOT**: AI 슬롭 (보라-시안 그라데이션, 네온 글로우, 글래스모피즘, 다크+보라)
- **Color**: 따뜻한 뉴트럴 (stone/warm gray) + 단일 accent (teal 또는 warm indigo)
- **Typography**: 한국어 우선 — Pretendard (본문) + 특색 있는 디스플레이 폰트
- **Layout**: 좌정렬 비대칭, 카드 남용 금지, 여백으로 호흡

## Differentiation
- "이것 AI가 만들었다고?" 라고 물어봐야 성공
- 한국 교육 앱 중 가장 깔끔한 인터페이스
- 수학 수식이 자연스럽게 녹아든 타이포그래피
