# 이차방정식과 인수분해

이 수업에서는 이차식 $x^2 + bx + c$를 인수분해해서 푸는 기본 방법을 다룹니다.

## 핵심 개념

1. 두 수의 곱이 상수항 $c$가 되고, 합이 일차항 계수 $b$가 되면 인수분해를 시도할 수 있습니다.
2. 예를 들어 $x^2 + x - 6$은 곱이 $-6$, 합이 $1$인 두 수를 찾으면 됩니다.
3. 그 두 수는 $3$과 $-2$이므로

$$
x^2 + x - 6 = (x + 3)(x - 2)
$$

4. 따라서 해는 $x = -3$, $x = 2$입니다.

## 판별식 메모

- 일반형 $ax^2 + bx + c = 0$에서 판별식은 $D = b^2 - 4ac$입니다.
- 판별식은 실근의 개수를 판단할 때 사용합니다.

## 학생에게 던질 질문 예시

- 곱해서 $-6$이 되고 더해서 $1$이 되는 두 수는 무엇인가요?
- 인수분해 뒤 각 인수를 0으로 두면 어떤 식이 되나요?
- 비슷하게 $x^2 + 5x + 6$은 어떻게 인수분해할 수 있나요?
