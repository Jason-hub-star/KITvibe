## 데모 흐름 (3분)

| 순서 | 시간 | 화면 | 행동 | 기대 결과 |
|------|------|------|------|----------|
| 1 | 0:00 | 랜딩 | "교사로 시작" 클릭 | 교사 업로드 페이지 |
| 2 | 0:15 | 교사 업로드 | "이차방정식" PDF 업로드 + 주제 입력 | "지식베이스 생성 완료" |
| 3 | 0:45 | 랜딩 | "학생으로 시작" 클릭 | 학생 질문 페이지 |
| 4 | 1:00 | 학생 질문 | 문제 사진 업로드 + "이 문제 어떻게 풀어요?" | AI 질문 제공 — Grill-Me (정답 아님) |
| 5 | 1:30 | 학생 질문 | "인수분해를 어떻게 해요?" 재질문 | 개념 설명 + 유사문항 |
| 6 | 2:00 | 학생 질문 | 미니 퀴즈 수행 | 오개념 회복 배지 |
| 7 | 2:15 | 교사 대시보드 | 교사로 전환 | 히트맵: "인수분해 80% 막힘" |
| 8 | 2:45 | 교사 대시보드 | 보충 설명 확인 | AI 추천: "합과 곱 관계 시각 복습" |

## 핵심 어필 포인트

1. **Grill-Me 질문 사다리:** AI가 질문을 던져 능동적 사고 유도 (Khanmigo 0.23SD)
2. **자료 근거:** "📚 수업자료 근거" 표시 → 환각 방지
3. **폐쇄 루프:** 학생 질문 → AI 질문 → 교사 대시보드 → 보충 개입
4. **30초 시작:** 로그인 없이 역할 선택만으로 즉시 사용
