# Style Guide — MathTutor AI

## 디자인 톤

- 깔끔한 에듀 SaaS
- 신뢰감 + 친절함 중심
- "장난감"이 아닌 "도구" 느낌

## 색상

- **기본:** zinc 기반 다크모드
- **Accent:** 1개 (blue 또는 indigo 계열 — 교육 신뢰감)
- **성공:** green (오개념 회복, 힌트 완료)
- **경고:** amber (근거 불확실 표시)
- **위험:** red (사용하지 않음 — 학습 맥락에서 부정적)

## 타이포그래피

| 용도 | 폰트 |
|------|------|
| UI 텍스트 | Geist Sans |
| 코드/수식/ID/타임스탬프 | Geist Mono |
| 수식 렌더링 | KaTeX |

## 컴포넌트 기준

- shadcn/ui 우선 사용
- Card + Table + Filters → 대시보드
- Sheet → 모바일 네비게이션 (필요 시)
- AlertDialog → 삭제 확인 (필요 시)

## 게임화 UX

| 허용 | 금지 |
|------|------|
| 힌트 진행도 (1/4→4/4) | 리더보드 |
| 오개념 회복 배지 | 코인/포인트 |
| 오늘 해결한 막힘 수 | 캐릭터/아바타 |
| 교사용 회복률 카드 | 경쟁 랭킹 |
